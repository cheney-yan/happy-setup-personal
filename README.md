# happy-setup-personal

Personal setup for self-hosting [Happy](https://github.com/slopus/happy) — a mobile/web client for Claude Code and Codex with end-to-end encryption.

Upstream Happy code lives in the `happy/` submodule. Personal configuration, provisioning scripts, and daemon setup live at the repo root.

## What is Happy?

Happy lets you run `happy claude` on your desktop and control it from your phone. It wraps Claude Code/Codex with a relay server so you get push notifications and can switch between devices instantly. All session content is end-to-end encrypted.

Open this repo in Claude Code — `CLAUDE.md` covers the full setup.
